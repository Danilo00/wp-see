# wp-see

Visualizzatore web per export chat WhatsApp.

## Struttura

- App Next.js in `web/`
- Chat e media in `web/chats/` (export WhatsApp con `_chat.txt` + allegati)

## Avvio rapido

```bash
cd web
npm install
npm run dev
```

Vedi [web/README.md](web/README.md) per configurazione e funzionalità.
