# wp-see

Visualizzatore web per export chat WhatsApp.

## Struttura

- Cartelle `WhatsApp Chat - …` con `_chat.txt` e allegati (root progetto)
- App Next.js in `web/`

## Avvio rapido

```bash
cd web
npm install
npm run dev
```

Vedi [web/README.md](web/README.md) per configurazione e funzionalità.
